## Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Fonts:** Cormorant Garamond · Syne · JetBrains Mono
- **Email:** Resend
- **Hosting:** Vercel

## Getting Started

```bash
npm install
cp .env.example .env.local
# Fill in RESEND_API_KEY and other vars
npm run dev
```
