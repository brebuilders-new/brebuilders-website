## RULES FOR IMAGE USE

1. **Every `<Image>` component MUST have alt and title from this registry.** No exceptions.
2. **Never leave alt="" unless image is purely decorative** (like a divider line).
3. **Download all images to /public/images/** during Phase 3 — don't rely on WP staying live.
4. **WebP format preferred** — use next/image with quality={85}.
5. **Lazy load all images below the fold** — loading="lazy" or next/image default.
6. **First Contentful Paint image** (hero thumbnail) must be `priority={true}` in next/image.
7. **Missing alt text items** (SVC-005 through SVC-015) — use the corrected values in this registry.
8. **The 619 Lakeview Dr project** has 37 images — only 1 currently in registry. Fetch remaining before building that project page.
